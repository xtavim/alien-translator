# Alien Translator Bot

A Discord bot that translates messages from Portuguese and Swiss German dialects to English. The bot forwards translated messages to a designated channel while preserving user role colors and message context.

## Environment Variables

Create a `.env` file in the project root with the following variables:

```
DISCORD_TOKEN=your_discord_bot_token_here
OPENAI_API_KEY=your_openai_api_key_here
